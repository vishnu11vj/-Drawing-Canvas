# 🎨 Drawing Canvas

A full-featured drawing application built on HTML5 Canvas.

## Features
- Tools: Pen, Eraser, Line, Rectangle, Circle, Fill (flood fill)
- Custom color picker and 8 preset colors
- Adjustable brush size and opacity
- Undo support
- Save drawing as PNG
- Dark canvas theme

## Built With
- HTML5 Canvas
- CSS3
- Vanilla JavaScript

## How to Run
Clone the repo and open `index.html` in your browser.
